== Description ==

Fixes the problem with Google Chrome v44 forcing SSL on WordPress websites. You should deactivate when Chrome updates to new version.



== Installation ==


1. Upload `chrome-ssl-fix.php` to the `/wp-content/plugins/chrome-ssl-fix/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What does this plugin do? =

It simply fixes your WordPress for Chrome v44 Users.
